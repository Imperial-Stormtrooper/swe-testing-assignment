# SWE Testing Assignment

## Project Description

This project implements a simple calculator written in Python.
The calculator supports the following operations:

* Addition
* Subtraction
* Multiplication
* Division

The project also contains automated tests written with the **pytest** framework.

---

# Installing Python

1. Go to the official Python website:

https://www.python.org/downloads/

2. Download the latest version of Python.

3. Run the installer.

4. IMPORTANT: check the option **Add Python to PATH**.

5. Finish installation.

Check installation in terminal:

~python --version~

---

# Installing pytest

Install pytest using pip:

~pip install pytest~

Verify installation:

~pytest --version~

---

# Running the Tests

Navigate to the project folder and run:

pytest

The test suite verifies that all calculator functions work correctly.
