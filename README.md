# SWE Testing Assignment

## Description
This project implements a simple calculator application and a test suite using pytest.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Installation

Clone the repository:

git clone [https://github.com/yourusername/swe-testing-assignment](https://github.com/Imperial-Stormtrooper/swe-testing-assignment.git)

Install dependencies:

pip install pytest

## Running Tests

pytest

## Testing Framework Research

The project uses pytest as the testing framework.

Advantages of pytest:
- simple syntax
- automatic test discovery
- powerful fixtures
- detailed error reporting
